JavaScript file: BiteBasket.js
CSS file: Style.css
PNG file: BiteBasket_logo.png

This project allows a user to write down an ingredient they would like in their food. The program then takes the ingredient, fetches an API, chooses a meal containing the ingredient at random and displays to the user the description of their meal and their order number. These details are shown in a Display Board, until the user enters their order number, which changes the state of the order from complete to incomplete.
